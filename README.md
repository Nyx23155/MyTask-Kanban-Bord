## English

# MyTask 📝

MyTask is a desktop application developed using C++ and the Qt 6 (QML) framework. 

> **Note:** The project is currently in early alpha (Work in Progress). The code and architecture are actively being developed; bugs and missing features are expected.

### 🛠 Tech Stack
* **Language:** C++
* **UI Framework:** Qt 6 (QML / Qt Quick)
* **Build System:** CMake
* **Database:** Qt SQL
* **Localization:** Qt Linguist (Ukrainian and Russian support)

### 📦 Installation & Run (For Users)
1. Go to the [Releases](../../releases) section and download the latest archive (e.g., `.7z` or `.zip`).
2. Extract the archive to any folder on your computer.
3. Run the `appMyTask.exe` file. No installation is required (Portable version with all dependencies included).

### 💻 Build from Source (For Developers)
If you want to build the project yourself, you will need **Qt 6.10+** (Quick, Sql, Network, Multimedia components) and **CMake**.

1. Clone the repository:
   ```bash
   git clone https://github.com/ВАШ_НИК/MyTask.git
   ```
2. Open the `CMakeLists.txt` file in Qt Creator.
3. Configure the build kit (MinGW or MSVC) and build the project.
